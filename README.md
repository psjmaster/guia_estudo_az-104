# 🌐 Preparação para o Exame AZ-104: Microsoft Azure Administrator

Este repositório documenta minha jornada de estudos para a certificação AZ-104, com anotações, comandos, links úteis e capturas de tela relacionadas às competências exigidas no exame.

---

## 🎯 Objetivo

Demonstrar conhecimento técnico e prático na administração de recursos do Microsoft Azure, incluindo redes, identidade, armazenamento, VMs e monitoramento, conforme os tópicos oficiais da certificação AZ-104.

---

## 📘 Tópicos de Estudo

### 1. Gerenciar identidades e governança no Azure (15–20%)
- Usuários, grupos e funções no Azure AD
- Controle de acesso com RBAC
- Políticas do Azure (Azure Policy)
- Gerenciamento de assinaturas e grupos

### 2. Implementar e gerenciar armazenamento (15–20%)
- Contas de armazenamento, blobs e file shares
- SAS, RBAC e replicação geográfica
- Backup e recuperação

### 3. Implantar e gerenciar recursos de computação do Azure (20–25%)
- VMs Windows/Linux e escalabilidade
- Azure App Services
- Templates ARM / Bicep

### 4. Configurar e gerenciar redes virtuais (20–25%)
- VNets, sub-redes, NSGs
- Peering, VPN, ExpressRoute
- Balanceadores de carga e DNS

### 5. Monitorar e fazer backup de recursos do Azure (10–15%)
- Métricas, alertas, Log Analytics
- Azure Monitor, Workbooks
- Azure Backup / Site Recovery

---

## 🛠️ Ferramentas e Recursos

- [Microsoft Learn – AZ-104](https://learn.microsoft.com/pt-br/certifications/exams/az-104/)
- Azure CLI e PowerShell
- Whizlabs / MeasureUp (simulados)
- Azure Sandbox e VS Code
